# Trade Journal Website

## Overview
A modular web application for journaling trades with timestamped clarity.  
Each entry becomes a legacy artifact for analyzing setups, breakdowns, and recovery pivots.

## Features (MVP)
- Trade entry form (date, ticker, entry/exit, notes)
- Journal archive with filters
- Export to CSV/JSON

## Tech Stack
- Frontend: HTML, CSS, JavaScript, Bootstrap
- Backend: Node.js + Express
- Database: SQLite (upgradeable to PostgreSQL)
- Visualization: Chart.js
